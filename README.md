# UDPBD Server

Istallation:
```
cd ~
git clone https://github.com/4gordi/udpbd-server.git
cd udpbd-server
make -f Makefile
chmod +x udpbd-server
wget -O setup.sh https://raw.githubusercontent.com/4gordi/udpbd-server/main/setup.sh
chmod 755 setup.sh
./setup.sh
```

Using:
```
udpbd-server /dev/sda1
```
